# GraphAn

GraphAn - це інтерактивний вебзастосунок для візуального моделювання та комплексного  аналізу графових структур, який поєднує функціонал графічного редактора для побудови графів різної конфігурації (орієнтованих, зважених, з петлями) з потужним обчислювальним модулем для автоматичного визначення метрик (матриць суміжності, зв’язності, хроматичних чисел, тощо) та візуалізації роботи класичних алгоритмів (Дейкстри, Флойда-Воршела, обходів), реалізованого на базі трирівневої архітектури з системою персоналізації користувачів для збереження та управління власними проєктами.

## Можливості

**Редактор графів**
- Додавання вершин, ребер, дуг і петель
- Перейменування вершин з перевіркою унікальності
- Вагові ребра з валідацією введення
- Масштабування та переміщення по полотну
- Збереження та відновлення проєктів

**Аналіз**
- Тип графа, кількість вершин та ребер
- Матриця суміжності та матриця інцидентності
- Список суміжності та степені вершин
- Найкоротший цикл, хроматичне число
- Число реберної та вершинної зв'язності
- Число незалежності та клікове число

**Алгоритми**
- алгоритм Дейкстра (для невід'ємних ваг)
- алгоритм Флойда-Воршела (для будь-яких ваг)
- алгоритм пошуку вглиб (DFS)
- алгоритм пошуку вшир (BFS)

Для кожного алгоритму є візуальна підсвітка на полотні та детальний протокол виконання.

## Ролі системи

- Гість (Guest) - має доступ лише до сторінок входу та реєстрації. Не може створювати або переглядати проєкти.
  
- Користувач (User) - може входити в 
систему та має повний доступ до функціоналу
